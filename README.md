## About Me

Hi, I'm Alina, a React and Full-Stack JavaScript Developer passionate about building modern, user-friendly web applications.

## Featured Projects

### ZNAC

Full-stack content platform built with React, Node.js, Express, and MongoDB.

Highlights:

- Public website, blog, projects showcase, photo gallery, and admin flows.
- React frontend with a custom Express API and MongoDB.
- Automated tests for frontend and backend.
- CI/CD with GitHub Actions, Docker, Nginx, GHCR, and AWS Lightsail.
